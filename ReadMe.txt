1. 兼容EUI与TidyPlate

    由于TidyPlate的代码实现机制原因，在设置中额外增加了一个勾选框，当使用TidyPlate的时候勾选上才能工作.
一般的, EUI,原生(包含整合插件大脚, 有爱), 如果没有开启TidyPlate, 默认不勾选.

2. 增加命令/fnp change 或者 /fnp ch 来切换启用和关闭。 于是可以做个宏, 一键打开或者关闭。默认关闭该功能。

3. 仅显模式和过滤模式

提供了2种模式，一种我称之为“仅显模式”，默认配置了一个"邪能炸药"。你可以用英文的分号来追加其他的名字。
另外一种是“过滤模式”，过滤模式就是添加过滤列表，能过滤某些姓名板，类似过滤图腾，猎人宝宝。
这2个模式是互斥的。
